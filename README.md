# yak8sui ☸️

> **Yet Another Kubernetes User Interface** — a pet project built with Go for learning purposes.

Inspired by [k9s](https://github.com/derailed/k9s) — a terminal UI for Kubernetes — but kept small and simple on purpose, as a learning exercise rather than a full-featured tool.

## Purpose of the Project
This project was born as a way to dive deep into:
* **Go (Golang)** patterns and application architecture.
* **Kubernetes API** (`client-go`) and how it handles real-time cluster state.
* **TUI (Terminal User Interface)** development in Go (`tview`).
